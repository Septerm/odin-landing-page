# odin-landing-page

## Introduction

This is the second project in The Odin Project’s Foundations course. Its purpose is to demonstrate the use of Flexbox for aligning and positioning elements to fulfill a landing page design brief.

## Tech Stack

- HTML
- CSS
- Git/GitHub
- VS-Code


## Skill Demonstrated 

- Utilized appropriate HTML elements and div containers to effectively structure page content for streamlined styling and positioning.

- Correctly applied CSS selectors and attributes to ensure the landing page visual style aligns strictly with the design brief.

- Mastered the use of Flexbox for the precise alignment and positioning of all page elements to meet specific design requirements.
